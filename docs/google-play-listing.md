# BoltShare Google Play listing

This copy is deliberately search-friendly without making claims the current product cannot support.

## App name (28/30 characters)

BoltShare: Secure File Share

## Short description (78/80 characters)

Send files by secure link or code with passwords, expiry and download tracking.

## Full description

Send files securely with BoltShare. Upload a file, create a private sharing link or transfer code, and let recipients download it from any modern device.

SECURE FILE SHARING

- Share files with a private link or easy transfer code
- Add password protection for sensitive downloads
- Set an expiry date so old links stop working
- Limit the number of permitted downloads
- Delete shared files and links when they are no longer needed

SIMPLE CROSS-DEVICE TRANSFERS

BoltShare works through the web, making it easy to send documents, photos and other supported files between Android phones, computers and other internet-connected devices. Recipients can open the link or enter the share code to receive the file.

TRACK EVERY SHARE

View transfer history, active links and download counts in one dashboard. File analytics help you see when a shared file has been downloaded.

BUILT FOR CONTROL

Your files are not listed publicly. You decide how each share is protected, when it expires and how many downloads are allowed.

HOW BOLTSHARE WORKS

1. Sign in securely.
2. Choose a file to upload.
3. Set password, expiry and download options.
4. Send the private link or transfer code.
5. Track downloads from your dashboard.

BoltShare is designed for quick, controlled file sharing when email attachments are inconvenient. Current transfers support files up to 4 MB.

## Data safety declaration checklist

The Console declaration should match the implementation before it is submitted:

- Data is collected and encrypted in transit.
- Account creation uses an email username with one-time-link authentication.
- Account and partial-data deletion URL: `https://boltshare.rcinc.app/account-deletion`
- Collected data includes email addresses, uploaded files and documents, transfer metadata, team data, IP addresses, user-agent data, and support messages.
- Data is used for app functionality, account management, security, support, and download analytics.
- Supabase, Bunny.net, Vercel, and Mailjet act as service providers.

Do not submit the declaration until both public policy URLs return HTTP 200.

## Claims to avoid

Do not claim unlimited or large-file transfer, Wi-Fi Direct, offline transfer, device-to-device transfer, or end-to-end encryption unless the product is changed and verified first.
